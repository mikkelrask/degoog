# Some basic rules

- Make sure to fork `develop` and branch off it. Pull requests MUST be directed to the `develop` branch.
- Read the various guides on how to [add a built-in engine](howto/ADD_NEW_ENGINES.md) or [Add a built-in plugin](howto/ADD_NEW_PLUGINS.md).
- If you use AI please make sure your code is reviewed, up to the coding standards set and is not overly engineered.
